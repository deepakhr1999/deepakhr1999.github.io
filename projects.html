Resumatic

A Hackathon winning Blockchain powered Dapp I developed with my team at the India Dapp Fest! This is a Dapp that stores evidence of a user's credentials as hashed records in the immutable blockchain ledger. This provides verification and value to one's resume and poses as a work-around for repeated background checks.

NowWeSeeYou

Automatic Attendance system using facial recognition using OpenCV. The software uses two cameras to recognize inflow and outflow of students, thereby monitoring the time spent by each student in the class. The information regarding attendance is stored at a portal built on django. I developed the code with my team at DevHack, an intra-college hackathon, ending up being placed second!.
